# NppVoicePlugin


## EN

📝 Boost your productivity! Dictate text, read selected text aloud, and send spoken or selected text directly to your local Ollama AI.


**Features:**
* Voice typing with speech recognition
* Text-to-speech
* Send spoken or selected text to your local Ollama AI
* Supports both English and Russian
* Seamless integration with Notepad++
* For detailed setup and usage instructions, see **Help** in the plugin menu.


🛠️ **Installation:**

Extract **NppVoicePlugin.zip** into the Notepad++ plugins folder.

Path: **Plugins → Open Plugins Folder** in the Notepad++ menu.

Example:
`C:\Program Files\Notepad++\plugins\NppVoicePlugin\`


After installation, open **Help** from the **NppVoicePlugin** plugin menu and read the documentation.


## RU

📝 Повысьте свою производительность с помощью голосового ввода и интеграции с ИИ! Диктуйте текст, читайте выделенный текст вслух и отправляйте произнесенный или выделенный текст непосредственно в локальный ИИ Ollama.


**Возможности программы:**
* Голосовой ввод с распознаванием речи
* Преобразование текста в речь
* Отправка произнесенного или выделенного текста в локальный ИИ Ollama
* Поддержка английского и русского языков
* Бесшовная интеграция с Notepad++
* Подробные инструкции по настройке и использованию см. в разделе «Справка» в меню плагина.


🛠️ **Установка:**
Распакуйте NppVoicePlugin.zip в папку плагинов Notepad++.

Путь: Плагины > Открыть папку плагинов в меню Notepad++.

Например: 
`C:\Program Files\Notepad++\plugins\NppVoicePlugin\`


Прочитайте Помощь в меню плагина NppVoicePlugin
