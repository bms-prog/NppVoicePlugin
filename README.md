# NppVoicePlugin
Boost your productivity! Dictate text, read selected text aloud, and send spoken or selected text directly to your local Ollama AI.

Повысьте свою производительность с помощью голосового ввода и интеграции с ИИ! Диктуйте текст, читайте выделенный текст вслух и отправляйте произнесенный или выделенный текст непосредственно в локальный ИИ Ollama.

===

Возможности программы:

 -Голосовой ввод с распознаванием речи

- Преобразование текста в речь

- Отправка произнесенного или выделенного текста в локальный ИИ Ollama

- Поддержка английского и русского языков

- Бесшовная интеграция с Notepad++

- Подробные инструкции по настройке и использованию см. в разделе «Справка» в меню плагина.
